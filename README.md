# Student Learning Management System
**CSEB5223 Group 2**
## Project Description 
The Student Learning Management System (SLMS) is a web-based system designed to support teaching and learning activities between students and lecturers. The system allows lecturers to manage course materials & assignments, while students can access learning resources, submit assignments, and view grades. Administrators manage users and course records.
## System Actors
* Student - View materials, submit assignments, view grades
* Lecturer - Upload materials, create assignments, assign grades
* Admin - Manage users and courses
## Main Features
* User authentication (Login/Logout)
* Course management
* Material upload and viewing
* Assignment creation and submission
* Grading system
* User management (Admin)
