# Student Learning Management System
**CSEB5223 Group 2**
## Project Description 
The Student Learning Management System (SLMS) is a web-based system designed to support learning activities for students. The system allows students to view available courses, manage their enrolled courses and maintain their student profiles.
## System Actors
* Student - View and manage course information, enroll in and view enrolled courses, manage student profile courses
## Main Features
* Add, search, edit, delete student profiles
* Add, search, edit, delete course profiles
* Enroll students into courses
* View all courses enrolled by a student
* View all students enrolled in a course
* Error handling for invalid inputs
